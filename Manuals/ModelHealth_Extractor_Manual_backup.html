Revit Model Health Extractor (Dynamo/Python)

เครื่องมือสกัดข้อมูลสุขภาพของโมเดล (Model Health) แบบอัตโนมัติสำหรับ Autodesk Revit 2024 ออกรายงานเป็น Excel ได้ทันที

🐍 โหมด Dynamo / Python

การติดตั้ง:

  1. เปิดโปรแกรม Revit 2024 และเข้าไปที่โปรเจกต์ของคุณ
  2. เครื่องมือนี้รันผ่าน Dynamo หรือ Dynamo Player โดยใช้ไฟล์ .py หรือ .dyn ที่ทีมงานเตรียมไว้ให้
  3. ตรวจสอบให้แน่ใจว่าในคอมพิวเตอร์ของคุณมีการติดตั้งไลบรารี openpyxl สำหรับ Python ไว้แล้ว (เพื่อจัดการไฟล์ Excel)

การใช้งาน:

  1. เปิดไฟล์โมเดล Revit ที่ต้องการตรวจสอบสุขภาพ
  2. รันสคริปต์ Model Health Extractor ผ่านหน้าต่าง Dynamo
  3. เมื่อมีหน้าต่างป๊อปอัปเด้งขึ้นมา ให้เลือก ตำแหน่งโฟลเดอร์ ที่ต้องการบันทึกไฟล์รายงาน Excel (.xlsx)
  4. กรอก Extraction Note (หมายเหตุหรือเฟสการทำงาน เช่น Pre-Submission, 50% DD)
  5. รอสคริปต์ทำงานจนเสร็จ ข้อมูลจะถูกเขียนและจัดกลุ่มลงในตาราง Excel โดยอัตโนมัติ

📊 ข้อมูลที่สามารถสกัดได้

  • 1
    Warnings & Errors
    รวบรวมจำนวนและรายการ Warnings ทั้งหมดที่มีในโปรเจกต์
  • 2
    Views & Sheets
    สกัดข้อมูล Views และ Sheets เพื่อเช็คความเรียบร้อยของการจัดหน้า
  • 3
    Linked Files & CAD
    รายการ Revit Links และ CAD Links (พร้อมเช็คสถานะ Pinned/Loaded)

🎯 จุดเด่นและข้อควรระวัง

📌
Creator Tracking:
สามารถติดตามชื่อผู้สร้าง (Worksharing User) ในรายการ Views, Sheets, CAD Links และ Line Styles ได้ทันที
📌
Append Data (บันทึกทับได้):
ระบบรองรับการเพิ่มข้อมูลลงในไฟล์ Excel ตัวเดิม เพื่อเปิด-ปิดดูประวัติการดึงข้อมูลเปรียบเทียบในแต่ละรอบ (Phase) ได้สะดวกขึ้น
⚠️
ข้อควรระวัง:
รองรับโมเดลที่อยู่บน Local, Revit Server และ Cloud Model ตรวจสอบว่าโมเดลได้ทำการ Sync ล่าสุดแล้วก่อนสกัดข้อมูลเพื่อความแม่นยำ
